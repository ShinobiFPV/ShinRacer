# Required tools

nircmd.exe — Required for game audio volume control (Car Stereo mixer, GAME channel).
Download from: https://www.nirsoft.net/utils/nircmd.html
Place nircmd.exe in this folder (resources/tools/nircmd.exe).

Without nircmd.exe, the GAME channel in the Car Stereo audio mixer will not
function — Settings -> Car Stereo will show "nircmd.exe not found" and the
GAME fader will be disabled. All other Car Stereo features (Spotify, YouTube
Music, Apple Music, local files, MUSIC and COMMS mixer channels) work without it.
